== このソフトについて ==

関連記事はこちら
http://ameblo.jp/handman-tk/entry-11814239410.html

Unity（無償版）でつくられたソフトです。
3DモデルはMetasequoiaで作りました。


== 動作環境 ==

Windowsマシン（Win7 32/64 bitで確認）
または
Mac OSXマシン（OSX 10.9.1 64bitで確認）


== ダウンロード ==

1. 以下のURLにアクセスします。

http://github.com/handman-tk/fujiko

2. Windowsの場合はtk-fujiko_win32.zip、Mac OSXの場合はtk-fujiko.app.zipをクリックします。

3. 「Raw」ボタン、または画面中央に表示される「View Raw」をクリックすると、ダウンロードが開始されます。

4. 保存したzipファイルを解凍します。


== 起動 ==

1. 解凍してできたtk-fujiko.exe (Windowsの場合)またはtk-fujiko.app (Mac OSXの場合)をダブルクリックします。

2. 最初に、開くウィンドウの大きさを選択してください。

3. 「Play!」ボタンをクリックすると起動します。

※Mac OSXの場合、「開発元が未確認のためアプリケーションを開けません」という警告が出ることがあります。これはこのソフトがアップルストアから正式にリリースされたものではないからです。警告ダイアログに表示される「詳細」をクリックすると、その説明が見られます。Controlキーを押しながらダブルクリックすることで、アプリケーションを起動できます。ハンドマンは悪意を持ったアプリを開発できるほどの開発力はないので、どうぞご安心ください。

== 操作方法 ==

・マウス右でドラッグ：　見る角度を変更します。

・マウスホイール回転：　ズームイン・ズームアウトをします。

・マウスホイールを押してドラッグ：　画面をスクロールします。

・スペースキー：　マシンをスタートさせます。一回押すと「用意！」、二回目で「スタート」です。

※Mac OSXの場合は、右クリックボタンとホイールのついたマウスをご利用ください。


== ご愛敬な点 ==

・表情はランダムですが、体の動きは気色悪い一定のパターンしかとりません。

・大笑いの表情でアゴが割れます（笑）アニメ設定の凡ミスです。

・結構頻繁に、台の表面が透明になり中身が見えてしまいます。なぜだか不明。

・終了ボタンはありません。終了するにはウィンドウを閉じてください。
